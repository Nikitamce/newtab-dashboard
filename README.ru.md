# NewTab Dashboard

**NewTab Dashboard** заменяет новую вкладку Firefox на персональный, минималистичный и полностью настраиваемый дашборд. Всё работает в браузере — без серверов, без аккаунтов, без API-ключей.

Доступно в **Firefox Add-ons Store**:
**https://addons.mozilla.org/es-ES/firefox/addon/newtab-dashboard/**

![My Dashboard screenshot](images/screenshot1.png)

---

## Что включено

Дашборд разделён на три колонки:

**Левая колонка — Sidebar**
- Часы и дата в реальном времени
- RSS-ленты с вкладками по источникам (Hacker News, The Verge, Ars Technica и любой другой фид)
- Мониторинг сервисов через **Uptime Kuma** (глобальный статус + сетка мониторов)

**Центральная колонка — Main**
- Поисковая строка с 8 настраиваемыми движками (Google, DuckDuckGo, Brave, Kagi, Perplexity…)
- Ссылки, организованные как **Разделы → Группы → Ссылки** с автоматическими иконками
- Последние видео с YouTube-каналов (без Google API key)

**Правая колонка — Panel**
- Погода в реальном времени через Open-Meteo (без API key)
- Месячный календарь с навигацией
- Цены криптовалют в реальном времени через CoinGecko (без API key)
- Постоянный блокнот

---

## Ключевые особенности

- **Без API-ключей** — Погода, крипта и видео YouTube работают без регистрации
- **Firefox Sync** — Настройки синхронизируются автоматически между устройствами
- **Автоматические иконки** — Ссылки предлагают иконку из Simple Icons при вводе имени
- **Настраиваемые обои** — URL изображения или готовый градиент, с контролем прозрачности, размытия, затемнения и тона
- **Импорт / Экспорт** — Сохраните и восстановите полную конфигурацию в JSON
- **Мультиязычность** — Интерфейс на русском, английском и испанском
- **Без зависимостей** — Vanilla JS, без фреймворков и бандлеров

## Установка

### Из магазина (рекомендуется)

Установите напрямую из Firefox Add-ons Store:
https://addons.mozilla.org/es-ES/firefox/addon/newtab-dashboard/

### Режим разработчика (из исходников)

1. Клонируйте или скачайте этот репозиторий.
2. Откройте Firefox и перейдите в `about:debugging`.
3. Нажмите **«Этот Firefox»** → **«Загрузить временное дополнение...»**.
4. Выберите файл `manifest.json` из папки проекта.

> Временные расширения отключаются при закрытии Firefox. Для постоянной установки используйте магазин.

## Кастомизация

Вся настройка делается через кнопку **⚙** в интерфейсе — без правки кода:

- Добавлять / переупорядочивать / удалять разделы, группы и ссылки
- Управлять RSS-лентами (с популярными пресетами)
- Добавлять крипто-активы (с пресетами)
- Добавлять YouTube-каналы по Channel ID или @handle
- Менять поисковый движок по умолчанию
- Применять обои (URL или градиент) и настраивать эффекты
- Переключать язык интерфейса
- Экспортировать и импортировать полную конфигурацию
- Смотреть информацию о проекте, версию и лицензию во вкладке **О программе**

Чтобы изменить город погоды, отредактируйте координаты в `dashboard.js` (ищите `open-meteo.com`).

## Технологии

| Слой | Детали |
|---|---|
| Хранение | `browser.storage.sync` (Firefox Sync) с fallback на `localStorage` |
| Погода | [Open-Meteo](https://open-meteo.com/) — бесплатно, без API key |
| Крипта | [CoinGecko API](https://www.coingecko.com/en/api) — бесплатно, без API key |
| YouTube | Прямой Atom XML feed — без Google API key |
| RSS | CORS proxy [rss2json](https://rss2json.com/) |
| Иконки | [Simple Icons](https://simpleicons.org/) CDN + favicon как fallback |
| Uptime | [Uptime Kuma](https://github.com/louislam/uptime-kuma) публичная Status Page |
